# SponsorBridge Landing Page

Welcome to the SponsorBridge project! This repository contains the code for a premium SaaS landing page designed to showcase the features and pricing of the SponsorBridge service.

## Project Structure

```
sponsorbridge
├── src
│   ├── app
│   │   ├── layout.tsx        # Main layout component
│   │   ├── page.tsx          # Main entry point for the landing page
│   │   └── globals.css       # Global CSS styles
│   ├── components
│   │   ├── Header.tsx        # Navigation and branding component
│   │   ├── Hero.tsx          # Main value proposition component
│   │   ├── Features.tsx      # Key features component
│   │   ├── Pricing.tsx       # Pricing plans component
│   │   ├── CTA.tsx           # Call-to-action component
│   │   └── Footer.tsx        # Footer component with links and info
│   ├── styles
│   │   └── variables.css     # CSS variables for theming
│   └── types
│       └── index.ts          # TypeScript types and interfaces
├── public
│   └── fonts                 # Custom font files
├── package.json              # npm configuration file
├── tsconfig.json             # TypeScript configuration file
├── next.config.js            # Next.js configuration file
├── tailwind.config.js        # Tailwind CSS configuration file
└── README.md                 # Project documentation
```

## Getting Started

To get started with the SponsorBridge project, follow these steps:

1. **Clone the repository:**
   ```
   git clone https://github.com/yourusername/sponsorbridge.git
   cd sponsorbridge
   ```

2. **Install dependencies:**
   ```
   npm install
   ```

3. **Run the development server:**
   ```
   npm run dev
   ```

4. **Open your browser:**
   Navigate to `http://localhost:3000` to view the landing page.

## Features

- Responsive design for optimal viewing on all devices.

- Engaging call-to-action to encourage sign-ups.
